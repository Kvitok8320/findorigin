# Как создать API ключ в Google Cloud Console

## Пошаговая инструкция:

### Шаг 1: Откройте раздел Credentials (Учетные данные)

**Способ 1 (через меню):**
1. В Google Cloud Console (https://console.cloud.google.com)
2. В левом меню найдите **"APIs & Services"** (APIs и сервисы)
3. Нажмите на **"Credentials"** (Учетные данные)

**Способ 2 (прямая ссылка):**
- Перейдите на: **https://console.cloud.google.com/apis/credentials**

### Шаг 2: Создайте API ключ

1. Вверху страницы нажмите кнопку **"+ CREATE CREDENTIALS"** (Создать учетные данные)
2. В выпадающем меню выберите **"API key"** (Ключ API)

### Шаг 3: Скопируйте ключ

1. Появится окно с вашим API ключом (он будет вида: `AIzaSy...`)
2. **Скопируйте ключ** - он понадобится позже
3. Нажмите **"CLOSE"** (Закрыть)

### Шаг 4: Ограничьте ключ (ВАЖНО для безопасности!)

1. Найдите созданный ключ в списке "API keys"
2. Нажмите на название ключа (или на иконку карандаша)
3. В разделе **"API restrictions"** выберите **"Restrict key"**
4. В выпадающем списке выберите **"Custom Search API"**
5. Нажмите **"SAVE"** (Сохранить)

**Почему это важно:**
- Ограничение ключа защищает его от использования другими сервисами
- Если ключ будет скомпрометирован, злоумышленники не смогут использовать его для других API

### Шаг 5: Проверка

После сохранения вы увидите:
- ✅ Ключ сохранен
- ✅ Ограничение применено

## Что дальше?

После создания API ключа нужно:

1. **Создать поисковую систему** на https://cse.google.com
   - Получить Search Engine ID
   
2. **Добавить ключи в Vercel:**
   - Зайдите в ваш проект на Vercel
   - Settings → Environment Variables
   - Добавьте:
     - `GOOGLE_API_KEY` = ваш API ключ
     - `GOOGLE_SEARCH_ENGINE_ID` = ваш Search Engine ID
   - Передеплойте проект

## Важные моменты:

- ⚠️ **Не публикуйте API ключ** в открытых репозиториях
- ⚠️ **Ограничьте ключ** только Custom Search API
- ⚠️ **Храните ключ в секрете** - это как пароль

## Если забыли скопировать ключ:

1. Вернитесь в "APIs & Services" → "Credentials"
2. Найдите ваш ключ в списке
3. Нажмите на иконку "Show key" (Показать ключ) или на название ключа
4. Скопируйте ключ
