# API включен, но ошибка 403 сохраняется

## Ситуация:
- ✅ Custom Search API включен (видно "API Enabled" с зеленой галочкой)
- ✅ API ключ создан и ограничен Custom Search API
- ❌ Все еще получаете ошибку 403

## Возможные причины:

### 1. API ключ создан в другом проекте

**Проверка:**
1. Зайдите в **"APIs & Services"** → **"Credentials"**
2. Найдите ваш API ключ
3. Нажмите на него
4. Посмотрите, в каком проекте он создан (будет указано вверху страницы)
5. Убедитесь, что это проект **"educationcursor"**

**Если ключ в другом проекте:**
- Создайте новый ключ в проекте "educationcursor"
- Или включите Custom Search API в том проекте, где создан ключ

### 2. API ключ не ограничен правильно

**Проверка:**
1. Зайдите в **"APIs & Services"** → **"Credentials"**
2. Найдите ваш API ключ
3. Нажмите на него
4. В разделе **"API restrictions"** должно быть:
   - Выбрано **"Restrict key"**
   - В списке выбран **"Custom Search API"**
5. Если нет - исправьте и сохраните

### 3. Нужно подождать дольше

После включения API или изменения ограничений может потребоваться **15-30 минут** для полной активации.

### 4. Проблема с квотами

**Проверка:**
1. Зайдите в **"APIs & Services"** → **"Quotas"**
2. Найдите "Custom Search API"
3. Проверьте, не превышены ли лимиты

### 5. API включен, но ключ не имеет доступа

Иногда нужно явно связать ключ с API:

1. Зайдите в **"APIs & Services"** → **"Credentials"**
2. Найдите ваш API ключ
3. Нажмите **"Edit"** (редактировать)
4. В разделе **"API restrictions"**:
   - Убедитесь, что выбрано **"Restrict key"**
   - Убедитесь, что в списке есть **"Custom Search API"**
   - Если нет - добавьте его
5. Нажмите **"Save"**
6. Подождите 10-15 минут

## Рекомендуемое решение:

### Вариант 1: Пересоздать API ключ

1. Удалите старый ключ
2. Создайте новый ключ в проекте "educationcursor"
3. Сразу при создании ограничьте его Custom Search API
4. Обновите `GOOGLE_API_KEY` на Vercel
5. Передеплойте проект
6. Подождите 15 минут

### Вариант 2: Использовать Bing Search API

Если проблема не решается, используйте Bing Search API:
- Более стабильный на Vercel
- 1000 запросов/месяц бесплатно
- Проще настройка
