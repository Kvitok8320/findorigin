# Как проверить, включен ли Custom Search API

## Быстрая проверка:

### Способ 1: Через Library (Библиотека)

1. В Google Cloud Console убедитесь, что выбран проект **"educationcursor"**
2. В левом меню нажмите **"APIs & Services"** → **"Library"**
3. В поиске введите: **"Custom Search API"**
4. Нажмите на **"Custom Search API"**
5. Посмотрите на кнопку:
   - Если написано **"ENABLE"** - API НЕ включен, нажмите и включите
   - Если написано **"MANAGE"** - API включен ✅

### Способ 2: Через Enabled APIs (Включенные API)

1. В левом меню нажмите **"APIs & Services"** → **"Enabled APIs"**
2. В поиске введите: **"Custom Search"**
3. Если видите **"Custom Search API"** в списке - API включен ✅
4. Если не видите - API не включен, нужно включить

### Способ 3: Прямая ссылка

Перейдите на:
**https://console.cloud.google.com/apis/library/customsearch.googleapis.com?project=educationcursor-484910**

(Замените `educationcursor-484910` на ваш Project ID, если отличается)

На этой странице:
- Если видите кнопку **"ENABLE"** - нажмите её
- Если видите кнопку **"MANAGE"** - API уже включен ✅

## Если API не включен:

1. Нажмите **"ENABLE"**
2. Подождите 2-3 минуты
3. Попробуйте снова отправить сообщение боту

## Если API включен, но ошибка сохраняется:

1. Проверьте переменные окружения на Vercel:
   - `GOOGLE_API_KEY` должен быть правильным
   - `GOOGLE_SEARCH_ENGINE_ID` должен быть правильным
2. Передеплойте проект на Vercel
3. Подождите 5-10 минут после включения API
