# Настройка Bing Search API (альтернатива Google)

## Почему Bing Search API:

- ✅ Более стабильный на Vercel
- ✅ 1000 запросов в месяц бесплатно (vs 100 в день у Google)
- ✅ Проще настройка
- ✅ Не требует биллинга для бесплатного тарифа

## Пошаговая инструкция:

### Шаг 1: Создание ресурса в Azure

1. Перейдите на **https://portal.azure.com**
2. Войдите в свой Microsoft аккаунт (или создайте бесплатный)
3. В поиске вверху введите: **"Bing Search v7"**
4. Нажмите на **"Bing Search v7"** в результатах
5. Нажмите **"Create"** (Создать)

### Шаг 2: Заполнение формы

1. **Subscription** (Подписка):
   - Выберите вашу подписку (или создайте бесплатную)

2. **Resource Group** (Группа ресурсов):
   - Создайте новую или выберите существующую
   - Название: например, "FindOrigin"

3. **Name** (Название):
   - Введите любое название (например: "findorigin-search")

4. **Pricing tier** (Тарифный план):
   - Выберите **"Free F1"** (1000 запросов/месяц бесплатно)

5. Нажмите **"Review + create"** (Просмотр и создание)
6. Нажмите **"Create"** (Создать)

### Шаг 3: Получение API ключа

1. После создания ресурса нажмите **"Go to resource"** (Перейти к ресурсу)
2. В левом меню найдите **"Keys and Endpoint"** (Ключи и конечная точка)
3. Скопируйте **"Key 1"** (Ключ 1)

### Шаг 4: Настройка в Vercel

1. Зайдите на https://vercel.com
2. Откройте ваш проект
3. Перейдите в **Settings** → **Environment Variables**
4. Добавьте переменную:
   - Key: `BING_API_KEY`
   - Value: ваш ключ из Azure
5. Нажмите **"Save"**
6. **Передеплойте проект** (Redeploy)

### Шаг 5: Тестирование

1. Отправьте сообщение боту в Telegram
2. Бот должен найти источники через Bing Search API

## Преимущества Bing Search API:

- ✅ Работает без проблем на Vercel
- ✅ Больше бесплатных запросов (1000/месяц)
- ✅ Не требует биллинга
- ✅ Проще настройка

## Лимиты:

- **Бесплатно:** 1000 запросов в месяц
- **Платно:** от $4 за 1000 запросов

## Если нужно больше запросов:

Можно использовать несколько ключей Bing Search API и переключаться между ними.
