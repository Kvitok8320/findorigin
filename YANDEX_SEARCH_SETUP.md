# Настройка Яндекс.Поиск API (рекомендуется для РФ)

## Почему Яндекс.Поиск API:

- ✅ **Не требует банковской карты** — регистрация полностью бесплатна
- ✅ **Работает в России** — специально разработано для российского рынка
- ✅ **Без геоблокировок** — российский сервис
- ✅ **Простая настройка** — не нужна подписка Azure или Google Cloud
- ✅ **Бесплатный доступ** — для личного использования

## Пошаговая инструкция:

### Шаг 1: Регистрация в Кабинете разработчика Яндекса

1. Перейдите на **https://developer.tech.yandex.ru**
2. Войдите в свой Яндекс аккаунт (или создайте новый)
3. Нажмите **"Получить ключ"** или **"Создать ключ"**

### Шаг 2: Создание API ключа

1. Заполните форму:
   - **Имя ключа**: например, "FindOrigin Bot"
   - **Сервис**: выберите **"API Яндекс.Поиска для сайта"**
   - Согласитесь с условиями использования
2. Нажмите **"Создать"** или **"Получить ключ"**
3. Скопируйте полученный API ключ (он будет показан только один раз!)

### Шаг 3: Настройка в Vercel

1. Зайдите на https://vercel.com
2. Откройте ваш проект
3. Перейдите в **Settings** → **Environment Variables**
4. Добавьте переменную:
   - Key: `YANDEX_API_KEY`
   - Value: ваш ключ из Кабинета разработчика
5. Нажмите **"Save"**
6. **Передеплойте проект** (Redeploy)

### Шаг 4: Тестирование

1. Отправьте сообщение боту в Telegram
2. Бот должен найти источники через Яндекс.Поиск API

## Преимущества Яндекс.Поиск API:

- ✅ Работает без проблем в России
- ✅ Не требует иностранной банковской карты
- ✅ Не требует подписки Azure
- ✅ Проще настройка, чем Google или Bing
- ✅ Бесплатный доступ

## Лимиты:

- **Бесплатно**: зависит от тарифа Яндекс.Поиска
- Уточняйте актуальные лимиты на https://yandex.ru/dev/site/api/
