# Как получить Search Engine ID на cse.google.com

## Пошаговая инструкция:

### Шаг 1: Откройте Google Custom Search Engine

1. Перейдите на **https://cse.google.com/cse/all**
2. Войдите в свой Google аккаунт (тот же, что используется для Google Cloud)

### Шаг 2: Создайте новую поисковую систему

1. Нажмите кнопку **"Add"** (Добавить) или **"+"** (плюс)
2. Откроется форма создания

### Шаг 3: Заполните форму

1. **Sites to search** (Сайты для поиска):
   - Введите: `*` (звездочка)
   - Это означает "искать везде в интернете"
   - Или можете указать конкретные сайты (например: `example.com`)

2. **Name** (Название):
   - Введите любое название (например: "FindOrigin Search" или "My Search Engine")

3. **Language** (Язык):
   - Выберите нужный язык (например: Russian или English)

4. Нажмите **"Create"** (Создать)

### Шаг 4: Найдите Search Engine ID

После создания:

1. Вы увидите список ваших поисковых систем
2. **Нажмите на название** созданной поисковой системы
3. Откроется страница настроек

**Способ 1 (через Setup):**
1. Перейдите во вкладку **"Setup"** (Настройка)
2. В разделе **"Basics"** (Основные)
3. Найдите поле **"Search engine ID"**
4. Скопируйте ID (он будет вида: `017576662512468239146:omuauf_lfve`)

**Способ 2 (через Details):**
1. На странице поисковой системы
2. Найдите раздел **"Details"** (Детали)
3. Там будет указан **"Search engine ID"**
4. Скопируйте ID

### Шаг 5: Скопируйте ID

ID будет выглядеть примерно так:
```
017576662512468239146:omuauf_lfve
```

Или короче:
```
017576662512468239146
```

**Важно:** Скопируйте весь ID, включая часть после двоеточия (если есть)

## Альтернативный способ (если уже есть поисковая система):

1. Перейдите на https://cse.google.com/cse/all
2. Найдите вашу поисковую систему в списке
3. Нажмите на нее
4. Перейдите в **"Setup"** → **"Basics"**
5. Найдите **"Search engine ID"**

## Что дальше?

После получения Search Engine ID:

1. **Добавьте в `.env`** (локально):
   ```
   GOOGLE_SEARCH_ENGINE_ID=ваш_id_здесь
   ```

2. **Добавьте в Vercel** (для продакшена):
   - Settings → Environment Variables
   - Key: `GOOGLE_SEARCH_ENGINE_ID`
   - Value: ваш ID
   - Save

3. **Передеплойте проект на Vercel**

## Важно:

- ⚠️ Search Engine ID отличается от API ключа
- ⚠️ Нужны ОБА: и API ключ, и Search Engine ID
- ⚠️ ID можно найти только после создания поисковой системы

## Если не видите ID:

1. Убедитесь, что вы нажали на название поисковой системы (не просто в списке)
2. Проверьте вкладку "Setup" → "Basics"
3. Попробуйте обновить страницу
